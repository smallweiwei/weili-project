#weili 后台管理系统

##项目介绍
###### This is a website for purchasing, selling and inventory. Its main functions are login module, administrator module, commodity module, sales module, store module, order module and warehouse module.

##错误提示码
    
        状态码                 描述
    
    登录模块
    
        -1001                   非法登录
        -1002                   用户名密码不能为空
        -1003                   管理员不存在
        -1004                   禁止登录
        -1005                   账号或者密码不正确
